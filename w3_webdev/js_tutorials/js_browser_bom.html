The Window Object

  • Supported by all browsers
  • Global variables belong to window object
  • Global functions are methods of the window object
  • This includes the document object window.document.getElementById("header") is the same as document.getElementById("header")

Imprtant properties of the window object include: window.innerWidth and window.innerHeight Some methods include window.open() , window.close(), window.moveTo() and window.resizeTo().

Location

  • window.location.href is the whole url
  • window.location.hostname is the base of the url
  • window.location.pathname is the path and the filename of the current page (the end of the url)
  • window.location.protocol is http or https.
  • window.location.assign() loads a new document

JS Cookies

One of the things we can do is:

document.cookie = "username=John Doe";

or

document.cookie = "username=lala lele; expires=Thu, 18 Dec 2013:00:00 UTC; path=/";

Then can read the cookies with

let x = document.cookie;

Returns all the cookies separated by ;.

To delete a cookie: document.cookie = “username=; expired=Thu, 01 Jan 1970 00:00:00 UTC; path=/;”; Have to do a special thing to get the way its going.

Have to have a JS function that searches the cookie string. It is includes:

function getCookie(cname) {
  let name = cname + "=";
  let decodedCookie = decodeURIComponent(document.cookie);
  let ca = decodedCookie.split(';');
  for(let i = 0; i <ca.length; i++) {
    let c = ca[i];
    while (c.charAt(0) == ' ') {
      c = c.substring(1);
    }
    if (c.indexOf(name) == 0) {
      return c.substring(name.length, c.length);
    }
  }
  return "";
}

Can then have a function that checks it:

function checkCookie() {
  let username = getCookie("username");
  if (username != "") {
   alert("Welcome again " + username);
  } else {
    username = prompt("Please enter your name:", "");
    if (username != "" && username != null) {
      setCookie("username", username, 365);
    }
  }
}
